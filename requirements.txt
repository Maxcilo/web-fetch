# Web Fetch Requirements

## Python Version
Python >= 3.8

## Core Dependencies
# 无需额外的 Python 包依赖
# 使用 Python 标准库

## System Dependencies
# curl - 用于 Jina Reader
# 通常系统自带，如果没有：
# Ubuntu/Debian: apt-get install curl
# CentOS/RHEL: yum install curl
# macOS: brew install curl

## External Scripts
# Scrapling - 需要单独安装
# 位置: ../scrapling/scrapling_fetch.py
# 或通过环境变量 SCRAPLING_PATH 指定

## Optional Dependencies (for future enhancements)
# requests>=2.28.0  # 替代 curl，提升性能
# aiohttp>=3.8.0    # 异步支持
# redis>=4.0.0      # 缓存支持

## Development Dependencies
# pytest>=7.0.0     # 测试框架
# black>=22.0.0     # 代码格式化
# pylint>=2.0.0     # 代码检查
# mypy>=0.900       # 类型检查

## Installation
# 基本安装（无需额外依赖）
# python3 web_fetch.py https://example.com

# 开发环境安装
# pip install pytest black pylint mypy
